# Terraform Azure Resource Group

A beginner Terraform project that creates an Azure Resource Group.

## Prerequisites

- Azure Subscription
- Terraform Installed
- Azure CLI

## Files

- provider.tf
- variables.tf
- main.tf
- output.tf

## Commands

terraform init
terraform validate
terraform plan
terraform apply

## Resources Created

- Azure Resource Group

## Learning Objective

Understand Terraform providers, variables, resources and outputs.
